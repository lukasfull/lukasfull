### 🔧 Protheus App Builder (VS Code Extension)  Open Source

Extensão para **Visual Studio Code** que automatiza a geração de arquivos `.app` para projetos Protheus que utilizam **PO-UI + ADVPL**.

👉 https://marketplace.visualstudio.com/items?itemName=lukasfull.protheus-app-generator

**Problema que resolve:**

No fluxo tradicional, após o build do front-end, é necessário:

- Executar `ng build`
- Acessar a pasta `dist`
- Ajustar manualmente a estrutura
- Compactar os arquivos (.app via WinRAR)
- Copiar para o projeto ADVPL
- Compilar via TDS ou VS Code

Esse processo é **repetitivo, manual e suscetível a erros**, principalmente em ambientes com múltiplos projetos.

---

**Solução proposta:**

Com o **Protheus App Builder**, todo esse fluxo é simplificado:

- Clique com o botão direito no projeto
- Selecione **"Gerar Protheus App Builder"**
- Escolha o diretório de saída
- Pronto — o `.app` é gerado automaticamente 🚀

---

**Benefícios:**

- ⚡ Redução de tempo no processo de build
- ❌ Menos erros manuais
- 📦 Padronização na geração de `.app`
- 🚀 Mais produtividade no dia a dia com Protheus
